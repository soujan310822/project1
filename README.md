## BMI calculator (Java)
This is my first beginner project using Java.
It calculates the Body Mass Index (BMI)
based on user input (height and weight)
and classifies it into categories.

## Features
-Takes height and weight as input
-Calculate BMI using formula 
-Displays BMI category:
   - Under Weight
   - Normal Weight
   - Over Weight
   - Obese
  
## Formula used
BMI=weight in (Kg)/height^2 in (m)
          or 
BMI=weight in (lbs)/height^2 in (inches)

## How to Run
1. Compile the program:
    javac BMI.java
2. Run the program
    java BMI
